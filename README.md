# sistem-informatic-shipping
Sistem informatic pentru integrarea si gestionarea serviciilor de productie si suport din cadrul unei companii de shipping.

Solutia vizeaza rezolvarea conflictelor dintre serviciile de 
productie:
- shipping (transport marfar)
- crew management (gestionarea personalului tehnic)
- tracking
- construction
suport:
- juridic
- financiar
- contabil
- management

Scopul sistemului informatic este de a centraliza accesul la servicii intr-o singura platforma, fiind mai usor de utilizat din partea angajatilor dar si mai usor de supervizat. 
In acest sens se vor minimiza resursele utilizate din punct de vedere al timpului, costului si efortului de munca, dezvoltand un flux de lucru optim pentru fiecare departament.
